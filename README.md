# 🤖 AI Mock Interview Platform

An AI-powered mock interview system that simulates real-world hiring interviews and evaluates candidates like a real recruiter.

## 🚀 Overview

This project uses Artificial Intelligence to conduct adaptive mock interviews based on a candidate’s resume and job description. It generates dynamic questions, evaluates answers, and provides detailed feedback to improve interview readiness.

## 🎯 Key Features

* Resume-based interview generation
* Job description analysis
* AI-driven adaptive question system
* Real-time answer evaluation
* Technical + behavioral + HR interviews
* Dynamic difficulty adjustment
* Hiring readiness score
* Detailed performance report

## 🧠 How It Works

1. Upload Resume
2. Paste Job Description
3. AI generates interview questions
4. Candidate answers in real-time
5. AI evaluates responses
6. Final report with scores & feedback

## 📊 Output Includes

* Overall Hiring Readiness Score
* Skill-wise Analysis
* Strengths & Weaknesses
* Improvement Suggestions
* Interview Performance Report

## 🛠 Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB
* OpenAI API

## 💡 Purpose

To help students and job seekers practice realistic interviews and improve their chances of getting hired through AI-driven evaluation.
